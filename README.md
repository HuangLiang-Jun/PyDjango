### API 回傳參數


| Key | 類型 | 描述 |   
| :-------------: | :-------------: | :-------------: |
| bank_id | int | 銀行id | 
| bank_name | str | 銀行名稱 |
| bank_code | str | 銀行代碼 |
| update_at | str | 匯率更新時間 |
| data      | list | 匯率 |

### data Info:

| Key | 類型 | 描述 |
| :-------------: | :-------------: | :-------------: |
| cn | str | 貨幣中文 |
| en | str | 貨幣英文 |
| cash_buying | str | 銀行現金買入 |
| cash_selling | str | 銀行現金賣出 |
| spot_buying | str | 銀行即期買入 |
| spot_selling | str | 銀行即期賣出 |
